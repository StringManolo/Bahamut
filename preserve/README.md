# Preserve

Here you will have some lists/output generated by Bahamut profiles/workflows

You can feed it to other modules creating a bash module like:
```bash
#!/usr/bin/env bash
# Name: Dummy Proxy Exporter
# Description: Exports small list domains for test
# Type: collector-domain
# Stage: 1
# Provides: httpproxy
# Install:
# InstallScope: shared

echo '{"bmop": "1.0", "module": "proxy-collector", "pid": 11082}
{"t": "batch", "f": "httpproxy", "c": 3}
108.162.198.103:80
108.162.198.104:80
108.162.198.109:80
{"t":"batch_end"}
{"t":"result","ok":true,"count":3}'
```

Or something like:
```bash
#!/usr/bin/env bash
# Name: Dummy Proxy Exporter
# Description: Exports small list domains for test
# Type: collector-domain
# Stage: 1
# Provides: httpproxy
# Install:
# InstallScope: shared

echo '{"bmop": "1.0", "module": "proxy-collector", "pid": 11082}
{"t": "batch", "f": "httpproxy"}'

$(cat ../../preserve/bigProxyList.txt)

echo '{"t":"batch_end"}
{"t":"result","ok":true}'
```
